Today's pattern is simple. If you want deep-target neuromodulation to be credible, stop choosing the montage by folklore.

The most relevant new paper is titled, OpTI-Mouse: Optimization for Targeted Temporal Interference Stimulation in the Mouse Brain. I picked it because it fixes an upstream source of sloppiness that quietly distorts a lot of preclinical stimulation work. The quick verdict is highly relevant.

Here is the core idea. The paper builds a mouse-specific temporal-interference planning pipeline. It constructs a head conductivity model, computes a lead field over a dense candidate electrode array, lets the user define a target region, and then optimizes the montage and current allocation against two competing goals: maximize target intensity and minimize spatial spread. That sounds technical and dry, but the point is very practical. Too much preclinical temporal-interference work still inherits empirical montages from older papers and then argues about downstream biology as if the targeting step were already competent.

The important result is that the gap versus empirical montages is not small. For the CA3-CA1 hippocampal target, the selected optimized strategy reaches ten point two nine volts per meter, compared with two point eight nine and four point four seven for the two literature-style baselines, while keeping focality in roughly the same range. For dentate gyrus, the optimized strategy keeps intensity about the same as baseline but tightens the half-max radius from three point nine nine millimeters to three point five four millimeters. Then the authors repeat the comparison in a separate Sim4Life reference model and the direction of the advantage survives. Good. That means the result is not just one lucky mesh.

The reason this matters is not that the paper proves temporal interference already works therapeutically. It does not. The reason it matters is that it shows how much room there still is for basic planning failure upstream. If target intensity can change several-fold under the same current budget just because the montage was optimized properly, then a lot of preclinical interpretation deserves more skepticism than the field usually gives it.

The second ranked note is titled, Transcranial temporal interference stimulation targeting the subthalamic region for motor symptoms in Parkinson's disease: a pilot, randomised, double-blind, sham-controlled crossover study. That note matters because it is one of the few temporal-interference papers that actually survives contact with a real blinded human clinical design. Thirty Parkinson participants received individualized targeting and active versus sham stimulation in the medication-off state. The signal is still preliminary, but it is much more serious than another field-simulation paper pretending to be translational evidence.

The third ranked note is titled, A first realization of reinforcement learning-based closed-loop EEG-TMS. That paper matters because it makes the controller earn the stimulation state online. Instead of hard-coding the supposedly correct mu-rhythm phase, it lets a reinforcement-learning agent search for the phase that increases or decreases corticospinal excitability for that participant and session. The big shared lesson with today's new note is that fixed parameter ritual is usually the wrong default.

The fourth ranked note is titled, Temporal interference stimulation for major depressive disorder: right lingual gyrus as a predictor. I still treat that one as adjacent rather than decisive, but it matters because it asks the right heterogeneity question. If deep-target stimulation helps some patients and not others, then patient state has to be part of the problem. Target naming alone is not enough.

So the pattern across the stack is becoming clearer. The montage should be optimized. The timing or control state should be discovered rather than assumed. And patient heterogeneity should be modeled instead of washed away in cohort averages.

The standing-interest checks were real but weaker. In the hypnosis lane, the cleanest recent hit was Hypnotic safety suggestions reduce cortisol awakening response and morning heart rate in daily life. That is better than the average hypnosis paper because it has physiology and a straightforward intervention design, but it is still a healthy-population stress-regulation result rather than a sharp interventional-psychiatry or state-control keep. In the CBT plus interventional psychiatry lane, the CBT-ENDURE randomized trial of CBT after esketamine still looks clinically useful for relapse prevention and care sequencing, but from the accessible abstract it still reads more like post-acute support architecture than a strong mechanism or targeting paper. And the psilocybin-assisted CBT pilot remains interesting background without yet settling whether CBT is a decisive mechanistic ingredient or mostly a competent therapeutic container.

The framing impact is straightforward. First, treat montage choice as an inverse-design problem, not a legacy recipe. Second, stop treating empirical preclinical stimulation setups as neutral baselines when they may be badly under-optimized. Third, remember that better field optimization is necessary but not sufficient, because neural and behavioral validation are still missing. And fourth, keep pushing parameter search upstream and downstream at once: upstream in montage design, downstream in online control and patient-specific response logic.

Final takeaway. Today's new paper matters because it removes one easy excuse from the neuromodulation workflow. If the field wants preclinical temporal-interference studies to be informative, it cannot keep acting as though inherited electrode layouts are good enough by default. The Parkinson pilot matters because human deep-target claims need blinded clinical contact. The closed-loop EEG-TMS paper matters because timing should be learned, not guessed. And the depression biomarker note matters because responders and nonresponders should not be treated as afterthoughts. The shared lesson is that montage, timing, and patient state all belong inside the problem.
