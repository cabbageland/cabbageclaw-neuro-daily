## Short overview

The most relevant paper today is **Phase-dependent stimulation response is shaped by the brain's dynamic functional connectivity**. Using connectome-based whole-brain modeling, the authors ask how the impact of a single pulse depends on both local oscillatory phase and the current large-scale functional-connectivity configuration. The useful claim is not just that stimulation is state dependent, which is already familiar. The stronger claim is that network-aware state descriptors can improve prediction of stimulation effects by up to about forty percent and that pulses can trigger whole-brain state switching. That is exactly the kind of paper that sharpens intervention logic rather than merely decorating it.

The second paper is **Bayesian time-history modeling enhances Parkinsonian motor state classification for adaptive deep brain stimulation**. In chronic at-home recordings from three Parkinson's disease patients with sensing-enabled DBS systems, the authors compare instantaneous discriminant classifiers against hidden-Markov time-history models for classifying motor state from cortical entrained gamma and subthalamic beta biomarkers. The main result is modest but real: adding time history improves hyperkinetic-state detection and smooths predictions without sacrificing real-time feasibility. The limitation is obvious, namely the tiny sample, but the control framing is still better than most adaptive-DBS decoding papers.

The third paper is **Personalized Virtual Brain Simulations and Vagus Nerve Stimulation Outcomes in Pediatric Epilepsy: A Multicenter Study From the CONNECTiVOS Collaboration**. Using preimplant diffusion and functional MRI from thirty-eight children with drug-resistant epilepsy, the authors fit subject-specific reduced Wong-Wang whole-brain models and compare responders with nonresponders after VNS. Nonresponders show worse model-based predictability of functional connectivity, while responders show stronger inferred inhibitory weights in thalamic, cingulate, and frontal nodes of the vagal afferent network. This is still model-dependent association rather than proof of mechanism, but it is a serious attempt to make responder heterogeneity computationally legible.

The fourth paper is **Intrinsic brain network dynamics modulated by neural stimulation to cerebellum**. Here the authors use resting-state fMRI before and after inhibitory repetitive transcranial magnetic stimulation to right cerebellar Crus I and analyze changes in dynamic community structure. Post stimulation, network flexibility rises and the dynamic patterns remain highly individual rather than collapsing into one canonical mode. This is not a clinical paper and not a targeting recipe, but it is useful for reminding us that stimulation can change large-scale integration and segregation properties in ways that simple average-connectivity summaries may miss.

## Most relevant paper

**Phase-dependent stimulation response is shaped by the brain's dynamic functional connectivity**

This gets top rank because it shifts the intervention question from, "what phase was the target in," to, "what coordinated network state was the brain in when the pulse landed." That is a much better control framing. If stimulation effects depend on transient distributed connectivity and can even push the brain into another global state, then a serious closed-loop system should estimate more than a local oscillatory scalar.

## Novelty / framing / baseline impact

- **For closed-loop stimulation logic:** the phase-dependent modeling paper raises the bar. Local phase alone may be too thin a state variable when distributed functional-connectivity state materially changes pulse effects.
- **For adaptive DBS:** the Bayesian motor-state paper is a useful baseline reminder that simple temporal models can add value before anyone reaches for giant black-box decoders.
- **For VNS personalization:** the pediatric TVB paper strengthens the case that responder heterogeneity should be framed as subject-specific network dynamics, not just demographic or syndrome-level stratification.
- **For network-neuroscience readouts:** the cerebellar rTMS paper says average pre-post connectivity changes can miss the more interesting effect, namely altered flexibility and modular switching.
- **For repo balance:** today leans intentionally toward network and computational papers because they improve intervention logic, but it still keeps translational neuromodulation central through adaptive DBS and pediatric VNS.

## One-paragraph takeaway

The useful pattern today is that better neuromodulation papers are getting more explicit about latent state. One paper says stimulation outcome depends on transient whole-network configuration, not just local phase. Another shows that adaptive DBS classifiers improve when they remember recent motor-state history instead of pretending each instant is independent. Another argues that VNS response heterogeneity may reflect whether a patient's network dynamics are even tractable to the intervention. And the cerebellar perturbation paper adds that stimulation can reshape modular flexibility in ways static summaries would hide. The common thread is simple: if you want better intervention logic, you need better state models.
