## Theme

Today is a **stop hard-coding the phase and make the controller earn it** day. The best new keep is **A first realization of reinforcement learning-based closed-loop EEG-TMS**, because it stops pretending the operator already knows which mu-rhythm phase matters and instead lets a real online controller search for it. The strongest ballast remains **Pre-stimulus brain states predict and control variability in stimulation responses**, **Phase-dependent stimulation response is shaped by the brain's dynamic functional connectivity**, and **A coupling model of transcranial magnetic stimulation induced electric fields to neural state variables**, because together they make the same point from different angles: state matters, network context matters, and the field-to-state bridge matters. The common lesson is not that reinforcement learning magically fixes neuromodulation. It is that fixed-parameter ritual is the wrong default.

## Short overview

The new preserve is the reinforcement-learning EEG-TMS preprint from Tuebingen and collaborators. Its real value is architectural. Instead of deciding in advance that one mu-rhythm phase is "high excitability" and another is "low excitability," the system chooses among eight phase bins online and learns from the paired-pulse MEP it actually produces. That is much closer to a real adaptive-control problem than most state-dependent TMS work. The paper is also refreshingly imperfect. The INCREASE condition works better than the DECREASE condition, the intended SMA-to-M1 facilitation does not cleanly show up, many sessions are excluded for poor mu-rhythm SNR, and a broader LTP-like effect appears 30 minutes later across conditions. Good. Those are the kinds of cracks that should be visible at this stage.

The surrounding context sharpens the interpretation. **Pre-stimulus brain states predict and control variability in stimulation responses** remains the stronger empirical argument that ongoing brain state is not nuisance noise but a usable control variable, and it does so at much larger scale even if our access there was only abstract-level. **Phase-dependent stimulation response is shaped by the brain's dynamic functional connectivity** keeps the controller honest by showing that local phase alone is under-specified once whole-brain state is allowed to matter. **A coupling model of transcranial magnetic stimulation induced electric fields to neural state variables** remains the right methods ballast because any control loop built on fake field-to-circuit coupling is decorative engineering.

## Most relevant paper

**A first realization of reinforcement learning-based closed-loop EEG-TMS**

This gets top rank because it is closer than most recent TMS papers to an actual controller rather than a stimulation protocol with a smarter slogan. The key result is not a giant effect size. It is the demonstration that the useful phase is session-specific enough to require online learning, and that a human EEG-TMS loop can at least start to learn it. The reason not to oversell it is equally important. The DECREASE side is weaker, the paired-pulse network logic is muddier than intended, and the delayed generic plasticity effect risks washing out the cleaner state-specific story. That combination makes it more valuable as a map of the problem than as a finished solution.

## Novelty / framing / baseline impact

- **For adaptive TMS:** stop benchmarking only against fixed-frequency or fixed-target protocols; benchmark against strong state-aware baselines and ask whether the controller truly learns something unstable enough that hard-coding fails.
- **For state-dependent stimulation:** predefining one "good" phase for everyone should now feel like a convenience assumption, not a serious mechanistic claim.
- **For network-level control:** a learned local phase target is still not enough if whole-brain FC state changes the effective response surface.
- **For stimulation modeling:** any future closed-loop stack that cannot justify how field geometry becomes neural-state input is still missing a core mechanistic layer.

## One-paragraph takeaway

The useful June nineteenth pattern is that neuromodulation gets better when it stops treating control variables as scripture. The new RL EEG-TMS paper matters because it makes the relevant phase something to discover online from feedback rather than something to declare. The older pre-stimulus-state paper matters because it shows there is real control-relevant signal upstream of the pulse. The dynamic-FC paper matters because local phase is only one slice of that signal, and the coupling-model paper matters because controllers still need a believable physics-to-state bridge. The shared message is simple: if the field wants adaptive neuromodulation, it has to learn the state, not merely name it.
